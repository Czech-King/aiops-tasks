# AI-Driven Cloud Security Threat Detection

An automated, event-driven security analysis and alerting system built with AWS GuardDuty, EventBridge, AWS Lambda, AWS Bedrock (`openai.gpt-oss-safeguard-120b`), and Slack.

High and medium severity security findings generated by GuardDuty are routed via EventBridge to a Python Lambda function. The Lambda filters for critical/high severity findings, passes the raw JSON finding to AWS Bedrock to perform intelligent risk assessment, and outputs a formatted Slack message with immediate remediation CLI commands.

---

## Architecture Diagram

```
[AWS GuardDuty] (Malicious activity)
       │
       ▼
[Amazon EventBridge] (Severity filter 4.0+)
       │
       ▼
 [AWS Lambda]
       │
       ├─► [Invoke AWS Bedrock (Risk Assessment & Remediation CLI Commands)]
       └─► [Post Slack Alert Message with Block Kit]
```
